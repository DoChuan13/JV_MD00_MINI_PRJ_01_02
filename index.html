Anh Musk ác đạn thật chứ, ngay sau khi mua lại Twitter ảnh ngay lập tức yêu cầu tất cả các dev tại Twitter in code ra giấy và anh ấy sẽ trực tiếp review code

Anh Musk ngoài làm playboy, lái cổ, lái coin thì ảnh vẫn là kỹ sư và biết code

Lần cuối tôi thấy người ta phải in code ra giấy là đầu những năm 2000 rồi, 2022 rồi ai còn bắt nhau in code ra giấy vầy hả anh Musk old school vãi nhái

—————-

“Theo bốn nhân viên hiện đang làm việc tại Twitter, các kỹ sư đã dành chiều thứ Sáu tại Twitter một cách nghiêm túc để in 30 ngày code của họ để chuẩn bị cho các cuộc họp riêng với Elon Musk và một số kỹ sư cấp cao của ông từ Tesla. Đặc biệt một số kỹ sư được yêu cầu chuẩn bị cho việc "code pairing" (ngồi lập trình cùng nhau) với Musk, trong đó họ sẽ ngồi với Elon và xem xét mã code cùng nhau.

Ngay sau buổi trưa, một trợ lý điều hành đã yêu cầu các kỹ sư bắt đầu chuẩn bị mã code để đưa cho Musk review. "Vui lòng in ra 50 trang mã code bạn đã thực hiện trong 30 ngày qua (nếu bạn chưa commit dòng code nào trong 30 ngày qua, thì bạn có thể tìm và in tối đa 60 ngày)", trợ lý viết trong một tin nhắn Slack. "Hãy sẵn sàng luôn cho việc mở code trên máy tính của bạn nếu ông ấy cần."
